System Improvements

Camera Text Recognition: When a camera is pointed towards text it will be able to recognize the text in images, allowing for copy-and-paste, look-up, and translation.

Cross-App Drag: Pictures can be dragged and dropped from app to another.

Suggested Widgets: This setting allows for the ability for widgets to be added to a Smart-Stack based on if the system believes the widget is relevant at that particular moment.

Improved Video Player: The default video player allows for different playback speeds.

Notification Customization: Notifications can be customized so that in addition to the default notification settings, notifications can all come at the same time, or filters can be applied to mute some notifications but allow others to come instantly.

Privacy Changes

App Privacy Report: If this report is activated, users will receiver a weekly report which determines all the times different 3rd-party apps attempted to access certain data.

Siri Privacy: When using Siri, instead of sending your request to a remote Apple server, Siri will convert your audio into words directly.

Hidden IP Address: Safari and Mail prevent known website trackers from accessing the user's IP address.

Hotspot Security: Now Hotspot's will utilize WPA3 security, which is more secure than previous version of WPA2.

New App Features

FacTime: New features in FaceTime include a grid view for group calls, voice isolation mode to remvoe background noises, FaceTime links, allowing Android users to join, and FaceTime can now use rear cameras

Messages: When multiple images are sent now, they will be sent in a photo library, or a stack, in stead of each image being sent similarly. Moreover, the user now has the ability to pin a contact in messages.

Photos: In the photos app, the user can now manually set the time date, and location of the photo.

Safari: Safari was completely redesigned, with the user interface changing. The tab and address bar were moved to the bottom. Users now have the ability to organize tabs and create different tab groups. Safari will also upgrade HTTP URLs to HTTPS.

Focus Mode

In the new focus mode, users will have the ability to set different states beyond simply Do Not Disturb, including work, sleep, and custom states. For each state, the user has the ability to customize which types of notifications would come through and which types of notification would be blocked. The user can also decide which specific contacts they want to to be notified from in each state.
